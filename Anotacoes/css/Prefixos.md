# Prefixos dos principais navegadores

- `-webkit-`
  - Aplicado no Chrome, Safari, versões mais recentes do Opera.
- `-moz-`
  - Aplicado no Firefox
- `-o-`
  - Versões antigas do Opera
- `-ms-`
  - Internet Explorer
