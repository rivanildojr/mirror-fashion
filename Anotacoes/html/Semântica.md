# HTML Semântico

É de suma importância utilizar as tags semânticas do HTML para a construção do seu site, possibilitando que o mesmo se torne acessivel a leitores de páginas e melhorando o SEO.

Algumas tags semânticas:

- <header>
- <section>
- <article>
- <nav>
- <aside>
- <main>
- <figure>
- <footer>
- <a>
- <em>
- <strong>
- <cite>
- <q>
- <time>
- <mark>
- <hgroup>
- <h1> ...
- ...
